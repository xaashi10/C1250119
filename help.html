Help & Instructions

How to use the Horizon Realty website

📋 Browsing Property Listings

To view all available properties, navigate to the Properties page using the navigation bar at the top of any page.

  • Scroll through the property images at the top of the page.
  • Below the images, you'll find a data table with all property details including name, location, price, bedrooms, bathrooms, size, and status.
  • The table rows highlight on hover and use zebra striping for easier reading.
  • Use the external links section to view properties on Google Maps or calculate mortgage costs.

📝 Filling Out the Booking Form

To book a consultation or search for properties, go to the Booking page.

  • Personal Information: Enter your full name, email, and phone number. Name and email are required.
  • Property Search: Select a preferred agent, property type, amenities, budget, and move-in date.
  • Comments: Add any special requests or questions.
  • Click Submit Booking Request to send your inquiry. Use Reset Form to clear all fields.

🖼️ Images & Navigation

The site uses a fixed navigation bar that stays at the top of the page as you scroll.

  • Home: Features a hero section and preview of featured properties.
  • Properties: Displays property images and a detailed data table.
  • Booking: Contains the search and consultation form.
  • Help: You are here — explains how to use the site.
  • About: Learn about the agency and its agents.

All property images have descriptive alt text for accessibility. Click any property card or "View details" link to go to the Properties page.
